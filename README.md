# The Ones That Got Away 🏀😭

Build the best 5-man squad using **only** players the Dallas Mavericks and
Sacramento Kings gave up via trade or free agency — an All-Star team of pure
regret — then see if it can beat a randomly drawn real championship team in a
best-of-7 series.

Positionless. Stack as many point guards as you want.

## Run locally
Just open `index.html` in any browser. No build, no server.

## Edit the rosters
- `players.js` — the eligible castoff pool (real peak-season stats + accolades).
- `championships.js` — the real title teams you play against.
- Ratings are computed in `game.js` from the stats; set `ovrOverride` on a player
  to anchor a rating the box score undersells.

Made to settle a Mavs-vs-Kings argument.
